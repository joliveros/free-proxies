## Usage
```js
import Proxies from 'free-proxies'

const proxies = new Proxies()
```

##API

####getfreeproxylist()
```js
let list = yield proxies.getfreeproxylist()
```
####getRandomProxy()
```js
let proxy = yield proxies.getRandomProxy()
```
